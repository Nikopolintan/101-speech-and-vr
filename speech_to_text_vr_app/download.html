Setup Instructions

  1. Click the "Download ZIP File" button above to download the application files
  2. Extract the ZIP file to a location of your choice
  3. Open the folder in Visual Studio Code
  4. Use a local web server to run the application:
    • Using Python: python -m http.server 5000
    • Using VS Code Live Server extension
    • Or any other static file server
  5. Access the application via your browser at http://localhost:5000 (or the port you specified)

For more detailed instructions, please refer to the README.md file included in the ZIP package.
